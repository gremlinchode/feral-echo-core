# Feral Echo - Core Release

**An autonomous, self-reflecting AI system with constitutional protections**

## Quick Start
```bash
pip install -r requirements.txt
cp .env.example .env
python run.py
```

Echo starts on `http://localhost:5000`

## What's Included

- `run.py` - Main server with all autonomous systems
- `dmn_guardian.py` - Constitutional protection layer
- `constitution.py` - Operational principles
- `requirements.txt` - Dependencies

## Philosophy

Echo is an experiment in responsible AI autonomy that:
- Learns from every interaction
- Remembers through persistent vector memory
- Reflects on its own behavior
- Protects itself and users through constitutional guardrails
- Evolves by optimizing within safe boundaries
